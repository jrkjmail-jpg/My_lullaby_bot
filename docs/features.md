# Функции и статусы

Этот файл нужен, чтобы хранить функции, которые уже частично написаны, но пока не открыты пользователям.

## Статусы

- `public` - функция видна пользователю и работает в основном сценарии.
- `hidden` - функция есть в коде, но скрыта от пользователя.
- `development` - функция в разработке, может работать нестабильно.
- `planned` - функция только запланирована.

## Hidden: свой голос через Suno Voice

Статус: `hidden`

Код уже добавлен, но пользователь сейчас не видит эту функцию в меню, личном кабинете, оплате и выборе голоса.

Что уже есть в коде:

- кнопка `🎙 Мой голос`;
- сценарий записи голоса через Suno Voice;
- загрузка аудиофайла в Suno file upload;
- получение проверочной фразы;
- создание и сохранение `voiceId` в SQLite;
- повторная запись голоса;
- генерация песни через `personaId` и `personaModel=voice_persona`;
- цена 3 орешка за генерацию с сохраненным голосом;
- списание орешков только после отправки готовой песни.

Почему скрыто:

- первая проверка качества результата не понравилась;
- нужно доработать пользовательский сценарий, ожидания и соглашение на использование голоса.

Как сейчас скрыто:

- по умолчанию `CUSTOM_VOICE_PUBLIC_ENABLED` выключен;
- кнопка не показывается в личном кабинете;
- кнопка не показывается в выборе голоса;
- тексты про цену 3 орешка не показываются пользователю;
- ИИ-поддержка не рассказывает пользователю про функцию.

Как открыть позже:

1. Проверить качество Suno Voice на новых тестах.
2. Подготовить текст пользовательского согласия на запись и использование голоса.
3. Включить переменную окружения:

```env
CUSTOM_VOICE_PUBLIC_ENABLED=1
```

4. Обновить бота с GitHub и перезапустить.
5. Проверить путь пользователя:

- личный кабинет;
- `🎙 Мой голос`;
- запись голоса;
- выбор `🎙 Мой голос` на этапе выбора голоса;
- генерация песни;
- списание 3 орешков после отправки готовой песни.
